## 📌 Sobre el Proyecto
Este repositorio contiene los archivos de mi entorno **Cloud**. Como estudiante de **Infraestructura de Redes Digitales y Ciberseguridad**, este espacio sirve para gestionar configuraciones y despliegues.

### 🛠️ Tecnologías utilizadas
* **Lenguajes:** PHP, HTML, CSS.
* **Redes:** Ubiquiti UniFi / Cisco.
* **Seguridad:** Hardening de servidores.
* **Herramientas:** Git, macOS Terminal (Zsh).

## 📂 Estructura de archivos
* `/uploads`: Gestión de archivos subidos.
* `verificacion.php`: Lógica de validación.
* `solucion_...jpg`: Evidencias de implementación.
